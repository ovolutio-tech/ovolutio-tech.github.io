Privacy Policy

Ovolutio Finance — Last updated: March 2026

Personal use only

1. Overview

Ovolutio Finance is a personal finance management application developed and used exclusively by its owner for private purposes. It is not a commercial product and is not offered to the public.

2. Data Collection

This application collects only the data you explicitly enter:

  • Bank account information (account name, IBAN, balance)
  • Financial transactions (amount, date, description, category)
  • Authentication credentials (email address, hashed password)

No data is collected automatically, and no analytics or tracking tools are used.

3. Open Banking (Enable Banking)

When you use the Enable Banking integration, this application connects to the Enable Banking API to retrieve your bank account data and transactions. This connection is initiated explicitly by you and requires your consent at your bank.

  • No banking credentials are stored by this application
  • Access tokens are used only during the import session
  • Data retrieved is stored locally in your personal database instance

4. Data Storage

All data is stored in a local PostgreSQL database running on your own infrastructure (Docker container). No data is sent to any remote server other than the Enable Banking API during the import process.

5. Data Sharing

No data is shared with any third party. This application is self-hosted and used exclusively by its owner.

6. Data Retention

Data is retained as long as you choose to keep it. You can delete your account and all associated data at any time from the application settings.

7. Your Rights (GDPR)

As the sole user and data controller, you have full control over your data, including:

  • Right of access — you can view all your data in the application
  • Right to rectification — you can edit any data at any time
  • Right to erasure — you can delete your account and all data
  • Right to portability — data can be exported via the database

8. Contact

For any questions regarding this privacy policy, please contact: github.com/ovolutio-tech
